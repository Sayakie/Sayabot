How to launch sayabot

1. Install Node.js higher than version 8
2. Follow that below a chapter ( Windows users must go to chapter B)

A.

```bat
$ npm install
$ npm start
```

B.

```bat
$ npm config set msvs_version 2015
$ npm install --global --production --vs2015 windows-build-tools
$ npm install
$ npm start
```

npm config get msvs_version
npm config get python
